Ambiente para inserção do conteúdo relacionado à disciplina Software Product: Analysis, Specification, Project &amp; Implementation

# Instruções para rodar o pbix na máquina local:

P1 - baixar o arquivo Dashboard_jira_AC2.pbix , localizado na pasta projeto_impacta/Power BI

P2 - baixar o arquuivo dados_jira_projectview.csv, localizado em dataset ( pode aparecer como "Conjunto de dados", por conta do tradutor do navegador)

P3 - baixar o arquuivo dados_jira_task.csv, localizado em dataset ( pode aparecer como "Conjunto de dados", por conta do tradutor do navegador)

P4 - De preferência, manter ambos os arquivos no mesmo diretório local

P5 - Alterar a fonte dos dados para a máquina local, direcionando para a pasta que salvou o arquivo csv
